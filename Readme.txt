【操作説明（デフォルト）】
　上下左右キー：カーソルの移動、マシンのステアリング
　Button1(Zキー)：決定、アクセル
　Button2(Xキー)：キャンセル、ブレーキ
　Button3(ESCキー)：プレイ中のゲームの中断

【インストール・アンインストール】
　圧縮ファイルを解凍し出てきたフォルダを適当な場所へ保存してください。
　フォルダ内の実行ファイルを起動することでゲームが起動します。
　アンインストールは、解凍したファイルを全て削除してください。

【ファイルについて】
_Debug/SmartPtr.hとSceneClass/iFuncSetter.hはIKD(http://marupeke296.com)様の物をお借りしております。
グラフィック、サウンドなどの全リソースファイルは部活動で同じく開発した参加した友人が製作しました。
それ以外の全ソースコードが私のオリジナルです。
